# Safe-Emotional-Prover (SEP)

A multi-modal agentic framework for **Automated Theorem Proving** that uses **Affective Reasoning** to manage proof-search frustration and implements **Conformal Policy Control** for safe, risk-tolerant proof exploration.

## Overview
SEP combines three cutting-edge techniques from recent ArXiv research:
1.  **T^3RL (Tool-Verified RL):** A self-evolution loop that uses external tools (Lean4) to upweight verified results in voting.
2.  **Nano-EmoX (Affective AI):** A cognitive hierarchy for emotional intelligence—detecting and managing search fatigue.
3.  **Conformal Policy Control (CPC):** A probabilistic regulator that provably enforces risk tolerance during exploration.
